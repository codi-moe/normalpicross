# Bad Game Idea

Imagine a regular picross (or nonogram, if you call it that) that goes totally normally.  
Except after some regular solves, you see a line indicator that totals to fifteen. And you were on a 10x board.

That might sound like it's impossible, but that just means you got to start thinking [outside the box](https://youtu.be/ZVy6LNBmlok?t=89).
